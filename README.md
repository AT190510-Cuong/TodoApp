## Available Scripts

In the project directory, you can run:

```npm install```

cài đặt dự án ( cài đặt mode_modules/libraries/framework)

```npm start```

Khởi chạy dự án tại thư mục chứa dự án

```json-server --watch db.json --port 3001```
